# Model Documentation – Fish Weight Prediction

---

## Model Type
Regression

---

## Algorithm
Linear Regression

---

## Input Features
- Species
- Length1

- Length3
- Height
- Width

---

## Output
- Fish Weight (grams)

---

## Train-Test Split
- Training: 80%
- Testing: 20%

- MSE
- MAE

- Sensitive to outliers
